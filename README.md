# Notify

A collaborative note-sharing platform where students can upload, discover, and rate course-related documents. Built with PHP and MySQL.

## Features

- **User authentication**: Secure registration and login with password hashing (bcrypt)
- **Note uploads**: Share PDF documents with course information and descriptions
- **Search & discovery**: Browse all uploaded notes organized by course
- **Rating system**: Rate and review other students' notes to surface quality content
- **User profiles**: Manage account information and view personal uploads
- **PDF viewer**: In-browser document viewing with PDF.js

## Tech Stack

- **Backend**: PHP with MySQLi
- **Frontend**: HTML5, CSS, Bootstrap 5.2
- **Database**: MySQL
- **Icons**: Font Awesome
- **PDF rendering**: PDF.js
- **Fonts**: Google Fonts (Montserrat, Ubuntu)
